## 问题与解决

### 问题 1: 编码错误
**现象**: `SyntaxError: invalid character '：'`  
**原因**: Node.js 文件被当作 Python 执行  
**解决**: 修改 index.js，直接调用 Python 脚本

### 问题 2: 图表公式错误
**现象**: `xl_range_formula() missing 1 required positional argument`  
**原因**: xlsxwriter 的 add_series 参数格式不正确  
**解决**: 使用 Excel 公式字符串格式代替数组格式
