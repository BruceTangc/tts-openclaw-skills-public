## 💡 使用建议

### 1. 缓存策略

**适合缓存**:
- ✅ 技术文档查询
- ✅ 教程搜索
- ✅ 通用知识

**不适合缓存**:
- ❌ 实时新闻（设置 use_cache: false）
- ❌ 股票价格
- ❌ 需要最新结果

**示例**:
```javascript
// 使用缓存（默认）
search("OpenClaw 教程")

// 禁用缓存（实时查询）
search({ 
  query: "最新 AI 新闻",
  use_cache: false 
})
```

### 2. 超时配置

```javascript
// 默认 10 秒超时
search(query)

// 自定义超时
search({ 
  query: "复杂查询",
  timeout: 20000  // 20 秒
})
```

### 3. 缓存管理

```bash
# 定期检查缓存
npm run cache:stats

# 遇到旧结果时清空缓存
npm run cache:clear
```
