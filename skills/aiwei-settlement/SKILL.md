# 艾威震源月底结算 - PDF 工单自动提取

从 ByWork PDF 工单压缩包自动提取数据，生成两种 Excel 表格：
1. **板材费用计算表** - 用于内部成本核算
2. **震源机械结算单** - 用于客户结算

## 快速开始

**直接发送压缩包给助手**，助手会自动：
1. 解压压缩包
2. 提取所有 PDF 数据
3. 从板材价格 Excel 读取当月采购价格
4. 生成两个 Excel 文件
5. 发送给您

## 价格自动更新

**自动读取**：脚本运行时自动从 `prices/` 目录读取最新的板材价格 Excel，无需手动传参。

```bash
python3 generate_both.py --pdf-dir <PDF文件夹>
```

**价格查找优先级：**
1. `--prices` 参数指定的文件（手动指定）
2. `prices/` 目录下最新的 `.xlsx` / `.xls` 文件（自动）
3. 脚本内硬编码价格（fallback 兜底）

**更新价格：** 把新月份的板材价格 Excel 放到 `prices/` 目录即可（覆盖或新增都行，自动取最新的）。

**自动识别逻辑：**
- 找到 "XX年X月份板材价格" 表，取最大月份作为当月
- 自动定位 "采购价格" 列
- 按材质（Q235/Q345/冷板）和厚度匹配
- 价格单位：元/吨 → 自动转换为 元/kg（÷1000）

## 生成的表格

### 1. 板材费用计算表
**用途**：内部成本核算

| 列 | 字段 | 公式/说明 |
|----|------|------|
| D | 材质 | 自动判断（Q235/Q345/镀锌板/SUS/耐磨板） |
| E-H | 长、宽、厚、数量 | 从 PDF 提取 |
| I | 单重 | 长×宽×厚×密度/1M |
| J | 合计重量 | 单重×数量 |
| L | 废料重量 | 合计×(1-利用率)×0.85 |
| M | 单价 | 按材质和厚度自动匹配（元/kg） |
| N | 板材价格 | 合计重量×单价 |

**材质判断规则**：
- `dxb-*.pdf` → 镀锌板
- `sus-*.pdf` → SUS 不锈钢
- `*Mn.pdf` → Q345 锰板
- 厚度≤2mm → Q235 冷板
- 其他 → Q235 开平板

**单价配置**：从板材价格 Excel 自动读取当月采购价格（支持 `--prices` 参数），无 Excel 时使用 fallback 价格：
- Q235 开平板：按减薄后厚度匹配价格表
  - 2.5mm: 3.883 元/kg | 2.75mm: 3.863 元/kg | 3.75mm: 3.783 元/kg
  - 4.75-11.75mm: 3.743 元/kg
- Q235 冷板：1.0mm 4.422 元/kg | 1.5mm 4.372 元/kg | 2.0mm 4.372 元/kg | 2.5mm 4.602 元/kg
- Q345 锰板：按减薄后厚度匹配价格表
  - 2.75mm: 3.912 元/kg | 3.75mm: 3.912 元/kg | 4.75mm: 3.862 元/kg
  - 5.75-9.75mm: 3.761 元/kg | 11.75mm: 3.831 元/kg
- 镀锌板：**5.0 元/kg**（固定价）
- SUS 不锈钢：**16.0 元/kg**（固定价）
- 耐磨板：**6.0 元/kg**（固定价）

**厚度规则**：
- Q235/Q345 标称厚度≥3mm 时，减薄 0.25mm 计算（如 5mm→4.75mm）
- 价格表按减薄后厚度匹配
- 价格表无对应厚度时，单价留空（如 14mm）

### 2. 震源机械结算单
**用途**：南通震源机械有限公司客户结算

| 列 | 字段 | 公式 |
|----|------|------|
| B | 厚度 mm | 标准规格 |
| E | 理论时间 | 从 PDF 提取累加 |
| F | 结算时间 | 理论时间×1.1 |
| H | 加工费总价 | 结算时间×6.33 |

**厚度规格**：1, 1.5, 2, 2.5, 3, 4, 5, 6, 8, 10, 12, 14, 16, 18, 20mm
