# 艾威震源月底结算 - PDF 工单自动提取

从 ByWork PDF 工单压缩包自动提取数据，生成两种 Excel 表格：
1. **板材费用计算表** - 用于内部成本核算
2. **震源机械结算单** - 用于客户结算

## 快速开始

**直接发送压缩包给助手**，助手会自动：
1. 解压压缩包
2. 提取所有 PDF 数据
3. 生成两个 Excel 文件
4. 发送给您

## 生成的表格

### 1. 板材费用计算表
**用途**：内部成本核算

| 列 | 字段 | 公式 |
|----|------|------|
| D | 材质 | 自动判断 |
| E-H | 长、宽、厚、数量 | 从 PDF 提取 |
| I | 单重 | 长×宽×厚×密度/1M |
| J | 合计重量 | 单重×数量 |
| L | 废料重量 | 合计×(1-利用率)×0.85 |
| N | 板材价格 | 合计×单价 |

### 2. 震源机械结算单
**用途**：南通震源机械有限公司客户结算

| 列 | 字段 | 公式 |
|----|------|------|
| B | 厚度 mm | 10 个标准规格 |
| E | 理论时间 | 从 PDF 提取累加 |
| F | 结算时间 | 理论时间×1.1 |
| H | 加工费总价 | 结算时间×6.33 |

**厚度规格**：1, 1.5, 2, 2.5, 3, 4, 5, 6, 10, 12mm
